大阪府マンション㎡単価ダッシュボード

1. 時系列推移

選択した市区町村・築年数・面積帯における㎡単価の四半期推移

2. 市区町村比較（最新四半期）

選択した築年数・面積帯における全72市区町村の最新四半期㎡単価。大阪市24区とその他48市町村で色分けし、各群内で降順に並べています。バーが描かれない区市町村はその条件でデータなし。

3. ヒートマップ（最新四半期）

選択した市区町村の最新四半期、築年数×面積帯マトリクス

データ出典: 国土交通省 不動産情報ライブラリ （XIT001 API: 成約価格情報 + 不動産取引価格情報）

「成約価格情報」と「不動産取引価格情報」とは？
成約価格情報
不動産流通機構（REINS）に登録された 仲介業者からの成約データ。 実際の成約価格そのものなので精度が高い。2021年から公開。 仲介を介さない自家取引はカバー外。
不動産取引価格情報
国土交通省が 買主にアンケートして集めた価格情報。 2006年から公開され、自家取引を含めカバー範囲が広い。 自己申告のため値段の端数丸めや回答漏れなどのノイズが含まれることがある。

両者は独立した別経路のデータで、本サイトでは中古マンションの取引のうち両系統を合算しています。 サンプル件数（n）が増えるぶん代表性が向上しますが、 n < 5 のセルは依然として参考値です。

※ サンプル件数が5件未満の値は信頼性が低い可能性があります（薄色マーカー）。
